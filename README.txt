[Installation]
Extract the files into the root of you StepMania 5 installation folder.

The accompanying simfile pack can be downloaded here: https://zenius-i-vanisher.com/v5.2/thread?threadid=9810

Place banner.png and jacket.png inside of your DDR XX song pack folder.

Be sure to delete any previous install of STARLiGHT if you're updating using the full theme pack.
Overwrite any files if you're using a hotfix file.

DancerVideos can be downloaded from here (1GB): https://mega.nz/#!7tpW2JxT!183BKx6l6W-ppFqKngFfQtwXVtaU17wU_-MDkWUVJFI
Place the video file inside the "DancerVideos" folder in each corresponding character's folder inside "SNCharacters"
